02 / Research

Dissertation Work

Primary Research · 2024–25

Deepfake Detection Using Convolutional Feature Extraction and Bidirectional Temporal Modeling

A novel CNN-BiLSTM architecture augmented with Squeeze-and-Excitation attention blocks for detecting AI-generated fake videos. Trained and evaluated on FakeAVCeleb v1.2 — a multimodal audio-visual dataset rebalanced to 4,000 clips at a 3:1 fake-to-real ratio.

92.83% Test Accuracy
91.22% Balanced Accuracy
90.59% Macro F1
60 Training Epochs

Architecture

  • 4-block SE-augmented CNN for spatial feature extraction
  • 2-layer Bidirectional LSTM for temporal sequence modeling
  • Focal Loss for handling class imbalance
  • Frame-level face extraction via MTCNN

Dataset

  • FakeAVCeleb v1.2 — multimodal fake video corpus
  • 4,000 clips rebalanced (3:1 fake-to-real)
  • 600-clip held-out test set
  • Video-level prediction via frame aggregation

Tooling

  • PyTorch · MTCNN · OpenCV
  • Google Colab (training + inference)
  • Matplotlib (publication-quality figures)
  • LaTeX (Mithibai College dissertation format)
